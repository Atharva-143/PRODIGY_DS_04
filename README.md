### Objective
To analyze and visualize sentiment patterns in social media data to understand public opinion and attitudes towards specific topics or brands.

---

### Dataset
Source: Twitter Entity Sentiment Analysis Dataset (Kaggle)

Description:
The dataset contains tweets related to various entities or brands, along with sentiment labels such as Positive, Negative, Neutral, and Irrelevant.

---

### Data Preprocessing
- Loaded the dataset using pandas.
- Assigned appropriate column names to the dataset.
- Removed rows with missing tweet text.
- Verified sentiment label distribution.

---

### Exploratory Data Analysis
- Analyzed the overall distribution of sentiment labels.
- Identified the most frequently mentioned entities.
- Visualized sentiment patterns for the top 5 entities.

---

### Visualizations
- Bar chart showing overall sentiment distribution.
- Count plot showing sentiment patterns across top 5 entities.

---

### Conclusion
The analysis provides insights into public opinion expressed on social media. The sentiment distribution and entity-wise sentiment patterns help in understanding how different topics or brands are perceived by users.
